# Robofriends - Front-End React App

## Overview of Project
Create-react-app project that uses React Redux to manage it's states, fetch APIs and allows the user search the fetched results.

## Resources
  - React.js
  - React Hooks
  - Redux
  - Tachyons
  - Robohash
  - Sublime Text 3

To run the project:
