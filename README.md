# Robofriends - Frontend React App

## Overview of Project
Interactive React project that allows you to search robots by their name and their contact cards will be displayed.

## Resources
  - React.js
  - Hooks
  - Redux
  - Tachyons
  - Robohash
  - Sublime Text 3

## Instructions
To run the project:

1. Clone this repository
2. Run `npm install`
3. Run `npm start`
